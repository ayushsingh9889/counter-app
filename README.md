# 🎯 Modern Counter App

A **production-ready, portfolio-worthy** counter application built with pure HTML, CSS, and JavaScript. Features include real-time statistics, dark mode, local storage persistence, keyboard shortcuts, and smooth animations.

## ✨ Features

### Core Features
- ✅ **Increment/Decrement** - Increase or decrease counter value
- ✅ **Reset Function** - Reset counter to zero
- ✅ **Custom Edit** - Set any number via input field
- ✅ **Real-time Statistics** - Track total changes, increments, decrements
- ✅ **Responsive Design** - Works on all devices (mobile, tablet, desktop)

### Bonus Features
- 🌙 **Dark Mode Toggle** - Seamless theme switching with persistence
- 💾 **Local Storage** - Saves counter state and preferences
- ⌨️ **Keyboard Shortcuts** - ↑ ↓ R E for power users
- 🔊 **Sound Effects** - Satisfying click sounds on actions
- 📊 **Statistics Dashboard** - Visual data tracking
- 🎨 **Glassmorphism UI** - Modern, professional design
- ♿ **Accessibility** - ARIA labels, keyboard navigation, focus states

## 🚀 Live Demo

Open `index.html` in any modern browser - no installation required!
