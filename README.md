# 🎯 Modern Counter App

A clean, feature-rich counter application built with pure HTML, CSS, and JavaScript. Perfect for learning web development or adding to your portfolio.

## ✨ Implemented Features

### Core Features
- ✅ **Increment Button** - Increase counter by 1
- ✅ **Decrement Button** - Decrease counter by 1  
- ✅ **Reset Button** - Reset counter to 0
- ✅ **Custom Edit** - Set any number via input field
- ✅ **Real-time Statistics** - Track total changes, increments, and decrements
- ✅ **Responsive Design** - Mobile, tablet, and desktop friendly

### Advanced Features
- 🌙 **Dark Mode Toggle** - Persistent theme switching with localStorage
- 💾 **Local Storage** - Saves counter state and statistics between sessions
- ⌨️ **Keyboard Shortcuts** - Quick access with arrow keys and letter keys
- 🔔 **Toast Notifications** - Real-time feedback for all actions
- ♿ **Accessibility** - ARIA labels, keyboard navigation, focus states
- 🎨 **Modern UI** - Glassmorphism design with smooth animations and gradients

## 🚀 Quick Start

1. **No Installation Required** - Just open `index.html` in your web browser
2. All features work offline - no server or dependencies needed
3. Data persists automatically in browser's local storage

## ⌨️ Keyboard Shortcuts

| Action | Key |
|--------|-----|
| Increase Count | ↑ Arrow Up |
| Decrease Count | ↓ Arrow Down |
| Reset Counter | R key |
| Focus Edit Input | E key |
| Submit Edit | Enter |

*Tip: Keyboard hint bar visible in bottom-left corner*
